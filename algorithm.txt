В таблице в ячейках есть либо адреса ячеек, либо числа. Требуется вычислить значение всех ячеек и вывести получившуюся
таблицу в консоль.
Вычисление значения ячейки заканчивается, когда мы получаем целое число. Вычисление может идти линейно, может ветвиться.

Для вычисления значения всех ячеек надо:
1. Получить таблицу. Вызвать для первой ячейки calculateCell (функция для вычисления значения ячейки).
2. Убрать знак "=", если необходимо. Если значение первого символа в ячейке - не число, получить значение следующей
ячейки по адресу из текущей ячейки. Вызвать calculateCell, передав значение следующей ячейки.
3. Если значение первого символа в ячейке - число, сохранить значение в ячейку, переданную в calculateCell.
4. Для случая выражения (ARG1 OP ARG2) надо:
4.1 Выявить, что это выражение по наличию арифметической операции из списка: +, -, *, /.
4.2 Выделить первый аргумент, второй аргумент, операцию. Вызвать calculateCell для первого и второго аргумента.
4.3 После получения чисел вместо первого и второго аргумента, выполнить арифметическое действие, и получившееся число
передать в calculateCell.
